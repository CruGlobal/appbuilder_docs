AppBuilder is a powerful platform for building, customizing, and managing web and mobile applications. 

Developers will find details of the architecture, as well as guides on how to make plugins to extend Appbuilder functionality
Builder and admins will also find detailed explanations of features
This documentation is meant to help you get the most out of AppBuilder! 

## Getting Started

- **Search:** Use the search bar at the top to quickly find pages, keywords, or topics.
- **Categories:** Each section groups related documentation for easier discovery, Such as development, user guides, mobile, and plugins
